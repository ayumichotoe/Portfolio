Overview

The TasteLab AI Toolkit is an advanced Vision-Language Model (VLM) powered system designed to revolutionize sensory data processing at the BUas TasteLab facility. This comprehensive solution integrates multiple data streams from 8 high-resolution cameras, wearable devices, and eye-tracking technology to provide real-time insights into consumer behavior and food preferences.

Built over an intensive 8-week development period, the system features a Flask-based dashboard with robust user authentication, experiment management capabilities, and sophisticated multimodal analysis tools. The project addresses the complex challenge of processing and interpreting diverse sensory inputs in real-time, enabling researchers to gain unprecedented insights into consumer interactions with food products.

Key Features

Multi-Camera Integration

Seamlessly processes data from 8 synchronized cameras for comprehensive behavioral analysis

Eye-Tracking Analysis

Advanced eye-tracking integration to understand visual attention patterns and preferences

Secure Authentication

Robust user authentication system ensuring data privacy and controlled access

Interactive Flask dashboard providing live visualization of sensory data and insights

Vision-Language Model integration for sophisticated multimodal data interpretation

3D Food Printer Support

Integration with 3D food printing technology for experimental product development

Technical Implementation

The system architecture leverages a microservices approach with Flask serving as the primary web framework. The backend processes multimodal data streams through a sophisticated pipeline that includes real-time video processing, sensor data aggregation, and machine learning inference.

Key technical challenges addressed include synchronization of multiple data streams, efficient processing of high-volume sensory data, and real-time visualization of complex analytical results. The implementation features advanced caching mechanisms, asynchronous processing capabilities, and scalable data storage solutions.
